### 将js改成面向对象的风格
- 先写出普通方法，再改成面向对象风格
- 普通方法遵循：
  - 尽量不要出现函数嵌套函数
  - 可以有全局变量
  - 把onload中不是赋值的语句放到单独的函数中
- 改造面向对象技巧：
  - 全局变量就是属性
  - 函数就是方法
  - onload中创建对象
  - 处理this指向，事件或定时器中this容易被修改
  ```
  var btn = document.getElementById("btnAdd");
  btn.onclick = function(){
      console.log(this);  //   btn
  }
  
  btn.onclick = add; //this=>btn
  
  add();  //this=>window
  
  btn.onclick = function(){
      add(); //this=>window
  }
  
  function add(){
      console.log(this); 
  }
  
 
  ```
